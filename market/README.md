# Duyler Market

### Get started

#### Install with included docker config

```shell
curl -L -O https://github.com/duyler/demo/archive/refs/heads/main.zip
```
```shell
unzip main.zip && mv demo-main duyler
```
```shell
cd duyler/market
```
```shell
make build
```
```shell
make up
```

**For show all available commands**

```shell
make help
```
